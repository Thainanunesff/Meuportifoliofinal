Projeto Taskflow

TaskFlow é uma aplicação web desenvolvida para ajudar usuários a organizarem suas tarefas de forma simples, eficiente e acessível. O sistema permite que qualquer pessoa crie uma conta, faça login e gerencie sua lista de afazeres com total controle sobre a criação, visualização, edição e exclusão de tarefas.

Tecnologias:

(HTML5 + CSS3 + JavaScript + Node.js + Express.js + CORS e dotenv + POSTGRESQL)
